Chapter 1 Basic Concepts

Data Flow Graphs

  • Node: In tensorflow, each node ==> instantion of an operation(>=0 inputs, >=0 outputs)
  • Edges:
    • Normal Edges: Carriers one operation's result(become the input of another op)
    • Special Edges: Special Edges: These edges are not data carriers between the output of a node (operator) and the input of another node. A special edge indicates a control dependency between two nodes. Let's suppose we have two nodes A and B and a special edges connecting A to B; it means that B will start its operation only when the operation in A ends. Special edges are used in Data Flow Graph to set the happens-before relationship between operations on the tensors.

  • Session

    When the client program has to establish communication with the TensorFlow runtime system, a session must be created. As soon as the session is created for a client, an initial graph is created and is empty. It has two fundamental methods

    • session.run
    • session.extend

Tensorflow programming model

Consider a simple example: multiply two integers.

steps:

  1. import 

    import tensorflow as tf

  2. define two vars a and b hold the data, and a b are called placeholder: 

    a = tf.placeholder("int32")
b = tf.placeholder("int32")

  3. placeholder allow us create our op and build our computation graph, without needing the data 

    y = tf.mul(a, b)

  4. manage the execution flow, create a session 

    sess = tf.Session()

  5. visualize the results, run our model, feed the data into the data flow 

    print(sess.run(y, feed_dict={a:2, b:5})

Computing gradients

y = 2*x2 

import tensorflow as tf
x = tf.placeholder(tf.float32)
y = 2 * x * x
var_grad = tf.gradients(y, x)
with tf.Session() as sess:
   grad_val = sess.run(var_grad, feed_dict={x: 1})
   print(grad_val)

Uniform distribution

import matplotlib.pyplot as plt
uniform = tf.random_uniform([100], minval=0, maxval=1, dtype=tf.float32)
with tf.Session() as sess:
    print(uniform.eval())
    plt.hist(uniform.eval(), normed=True)
    plt.show()

Normal Distribution

same as uniform distribution 

norm = tf.random_normal([100], mean=0, stddev=2)
with tf.Session() as sess:
   plt.hist(norm.eval(), normed=True)
   plt.show()
